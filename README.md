# GeoSharpStratosyn

Interactive **Geometry** course — a Stratosyn build for **Stephen Salas** (homeschool),
taught three sessions a week on a **Concept → Build → Prove** rhythm across 40 weeks.

This repository is the **public GitHub Pages site** for the course's self-contained
HTML interactives — the projector **boards**, the print-and-self-check **worksheets**,
the **checkpoints**, and the geometry **sims** (all built on a shared `geo.js` toolkit).

**Not in this repo, by design:** the Math-U-See Geometry Workbook and Test scans are
copyrighted and stay local only (see `.gitignore`). Teacher answer keys inside the
worksheets are passphrase-gated.

Built and maintained by Colton H. Sharp.
